Quick Setup Guide

1 Create Google Analytics 4 Account

Go to Google Analytics and create a new GA4 property for flightandtrace.com

2 Get Your Measurement ID

Navigate to: Admin â†’ Data Streams â†’ Web â†’ Your Stream

Copy your Measurement ID (starts with G-)

3 Add to All HTML Pages

Add this code to the <head> section of index.html, live.html, and track.html:

<!-- Google Analytics 4 -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-YOUR_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-YOUR_ID');
</script>
4 Add to Environment Variables

In Vercel Dashboard, add:

GA_MEASUREMENT_ID=G-YOUR_ACTUAL_ID

Custom Events to Track

These events are already implemented in the code:

  • Flight Searches: Track every flight search
  • Subscription Purchases: Track conversions and revenue
  • Feature Usage: Track which features are most popular
  • User Engagement: Time on site, pages viewed

Files to Update

Add the GA4 tracking code to these files:

  • /public/index.html - Homepage
  • /public/live.html - Live tracker
  • /public/track.html - Flight search

Example implementation:

// Track flight search
gtag('event', 'search', {
    search_term: 'AA123',
    search_type: 'flight_number'
});

// Track subscription
gtag('event', 'purchase', {
    currency: 'USD',
    value: 7.99,
    items: [{
        item_name: 'Premium Subscription'
    }]
});

Verification

After adding the code:

  1. Deploy to Vercel
  2. Open Google Analytics Realtime view
  3. Visit your site
  4. You should see yourself as an active user

Recommended GA4 Configuration

  • Enable Enhanced Measurement (automatic tracking)
  • Set up conversion events for subscriptions
  • Create audiences for free vs paid users
  • Set up custom dimensions for subscription tier
  • Configure data retention to 14 months
